

# or

Performs a bitwise OR operation between the two values.

```kotlin
infix inline fun Byte.or(other: Byte): Byte(source)
```

```kotlin
val a: Byte = 0b0101_0101
val b: Byte = 0b0011_1000
val result: Byte = a or b
println("Result: $result")
```
