

# scanIndexed

Returns a sequence containing successive accumulation values generated by applying operation from left to right to each element, its index in the original sequence and current accumulator value that starts with initial value.

```kotlin
fun <T, R> Sequence<T>.scanIndexed(initial: R, operation: (index: Int, acc: R, T) -> R): Sequence<R>(source)
```

```kotlin
val numbers = sequenceOf(1, 2, 3, 4)
val cumulative = numbers.scanIndexed(0) { index, acc, value ->
    acc + value + index
}.toList()
println(cumulative) // [0, 1, 4, 9, 16]
```
