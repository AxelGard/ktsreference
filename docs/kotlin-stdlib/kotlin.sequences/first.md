

# first

Returns the first element.

```kotlin
fun <T> Sequence<T>.first(): T(source)
```

```kotlin
val numbers = sequenceOf(10, 20, 30, 40)
val firstNumber = numbers.first()
println(firstNumber) // prints 10
```
