

# log10

Computes the common logarithm (base 10) of the value x.

```kotlin
expect fun log10(x: Double): Double(source)
```

import kotlin.math.log10

fun main() {
    val value = 100.0
    val result = log10(value)
    println("log10($value) = $result")
}
