

# sin

Computes the sine of the angle x given in radians.

```kotlin
expect fun sin(x: Double): Double(source)
```

```kotlin
import kotlin.math.sin

fun main() {
    val angleRadians = Math.PI / 4   // 45 degrees
    val result = sin(angleRadians)
    println("sin($angleRadians) = $result")
}
```
