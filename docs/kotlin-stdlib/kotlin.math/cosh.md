

# cosh

Computes the hyperbolic cosine of the value x.

```kotlin
expect fun cosh(x: Double): Double(source)
```

fun main() {
    val x = 1.0
    val result = kotlin.math.cosh(x)
    println("cosh($x) = $result")
}
