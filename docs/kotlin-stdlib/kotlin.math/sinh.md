

# sinh

Computes the hyperbolic sine of the value x.

```kotlin
expect fun sinh(x: Double): Double(source)
```

import kotlin.math.sinh

fun main() {
    val x = 1.0
    val y = sinh(x)
    println("sinh($x) = $y")
}
