

# sqrt

Computes the positive square root of the value x.

```kotlin
expect fun sqrt(x: Double): Double(source)
```

```kotlin
import kotlin.math.sqrt

fun main() {
    val number = 9.0
    val root = sqrt(number)
    println("Square root of $number is $root")
}
```
