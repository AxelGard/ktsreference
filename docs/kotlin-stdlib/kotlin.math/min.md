

# min

Returns the smaller of two values.

```kotlin
inline fun min(a: UInt, b: UInt): UInt(source)
```

```kotlin
import kotlin.math.min

fun main() {
    val a: UInt = 10u
    val b: UInt = 20u
    val smaller = min(a, b)
    println("The smaller value is $smaller")
}
```
