

# atanh

Computes the inverse hyperbolic tangent of the value x.

```kotlin
expect fun atanh(x: Double): Double(source)
```

```kotlin
import kotlin.math.atanh

fun main() {
    val x = 0.5
    val result = atanh(x)
    println("atanh($x) = $result")
}
```
