

# ln

Computes the natural logarithm (base E) of the value x.

```kotlin
expect fun ln(x: Double): Double(source)
```

```kotlin
import kotlin.math.ln

fun main() {
    val number = 2.71828
    val naturalLog = ln(number)
    println("The natural logarithm of $number is $naturalLog")
}
```
