

# acosh

Computes the inverse hyperbolic cosine of the value x.

```kotlin
expect fun acosh(x: Double): Double(source)
```

```kotlin
import kotlin.math.acosh

fun main() {
    val x = 2.0
    val y = acosh(x)
    println("acosh($x) = $y")
}
```
