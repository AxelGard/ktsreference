

# expm1

Computes exp(x) - 1.

```kotlin
expect fun expm1(x: Double): Double(source)
```

```kotlin
import kotlin.math.expm1

fun main() {
    val x = 0.5
    val result = expm1(x)
    println("expm1($x) = $result")   // Output: expm1(0.5) = 0.6487212707001282
}
```
