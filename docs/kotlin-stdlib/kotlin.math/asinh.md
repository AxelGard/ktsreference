

# asinh

Computes the inverse hyperbolic sine of the value x.

```kotlin
expect fun asinh(x: Double): Double(source)
```

```kotlin
import kotlin.math.asinh

fun main() {
    val x = 2.5
    val result = asinh(x)
    println("asinh($x) = $result")
}
```
