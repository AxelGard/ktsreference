

# absoluteValue

Returns the absolute value of this value.

```kotlin
expect val Double.absoluteValue: Double(source)
```

```kotlin
fun main() {
    val negative = -42.7
    val positive = negative.absoluteValue
    println("Absolute value of $negative is $positive")
}
```
