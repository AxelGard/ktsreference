

# parseInt

Use toInt() instead.

```kotlin
external fun parseInt(s: String): Int(source)
```

```kotlin
import kotlin.js.parseInt

fun main() {
    val str = "42"
    val num = parseInt(str)
    println("Parsed number: $num")
}
```
