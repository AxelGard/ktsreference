

# nextUInt

Gets the next random UInt from the random number generator.

```kotlin
fun Random.nextUInt(): UInt(source)
```

```kotlin
import kotlin.random.Random

fun main() {
    val randomUInt = Random.nextUInt()
    println("Generated UInt: $randomUInt")
}
```
