

# nextULong

Gets the next random ULong from the random number generator.

```kotlin
fun Random.nextULong(): ULong(source)
```

```kotlin
import kotlin.random.Random

fun main() {
    val randomULong: ULong = Random.Default.nextULong()
    println("Random ULong: $randomULong")
}
```
