

# toUpperCase

Warning since 1.5

```kotlin
expect fun Char.toUpperCase(): Char(source)
```

```kotlin
fun main() {
    val lowerChar: Char = 'k'
    val upperChar: Char = lowerChar.toUpperCase()
    println(upperChar) // Output: K
}
```
