

# setCharAt

Warning since 1.3

```kotlin
inline fun StringBuilder.setCharAt(index: Int, value: Char)(source)
```

```kotlin
val sb = StringBuilder("Hello")
sb.setCharAt(0, 'J')   // change 'H' to 'J'
println(sb)            // prints "Jello"
```
