

# drop

Returns a subsequence of this char sequence with the first n characters removed.

```kotlin
fun CharSequence.drop(n: Int): CharSequence(source)
```

```kotlin
val text = "Hello, World!"
val result = text.drop(7)   // "World!"
println(result)
```
