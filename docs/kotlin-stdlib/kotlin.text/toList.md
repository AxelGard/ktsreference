

# toList

Returns a List containing all characters.

```kotlin
fun CharSequence.toList(): List<Char>(source)
```

```kotlin
val str = "Hello, Kotlin!"
val charList = str.toList()

println(charList) // [H, e, l, l, o, ,,  , K, o, t, l, i, n, !]
```
