

# toUByte

Parses the string as a signed UByte number and returns the result.

```kotlin
fun String.toUByte(): UByte(source)
```

```kotlin
fun main() {
    val numberStr = "200"
    val number: UByte = numberStr.toUByte()
    println(number) // 200
}
```
