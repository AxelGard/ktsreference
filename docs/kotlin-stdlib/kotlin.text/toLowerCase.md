

# toLowerCase

Warning since 1.5

```kotlin
expect fun Char.toLowerCase(): Char(source)
```

```kotlin
val upper: Char = 'K'
val lower: Char = upper.toLowerCase()   // 'k'
println("Upper: $upper  →  Lower: $lower")
```
