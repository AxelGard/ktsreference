

# decapitalize

Warning since 1.5

```kotlin
expect fun String.decapitalize(): String(source)
```

```kotlin
fun main() {
    val original = "HelloWorld"
    val decapitalized = original.decapitalize()
    println(decapitalized) // prints "helloWorld"
}
```
