

# max

Returns the largest character.

```kotlin
@JvmName(name = "maxOrThrow")fun CharSequence.max(): Char(source)
```

```kotlin
val sentence = "Hello, World!"
val maxChar = sentence.max()
println("The largest character is: $maxChar")
```
