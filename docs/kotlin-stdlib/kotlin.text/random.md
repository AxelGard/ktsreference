

# random

Returns a random character from this char sequence.

```kotlin
inline fun CharSequence.random(): Char(source)
```

```kotlin
fun main() {
    val sampleText = "Kotlin is awesome!"
    val randomChar = sampleText.random()
    println("A random character from the string: $randomChar")
}
```
