

# count

Returns the length of this char sequence.

```kotlin
inline fun CharSequence.count(): Int(source)
```

```kotlin
fun main() {
    val text = "Hello, Kotlin!"
    val length = text.count()
    println("The length of \"$text\" is $length.")
}
```
