

# min

Returns the smallest character.

```kotlin
@JvmName(name = "minOrThrow")fun CharSequence.min(): Char(source)
```

```kotlin
fun main() {
    val text = "kotlin"
    val smallest = text.min()   // returns 'i'
    println("The smallest character is: $smallest")
}
```
