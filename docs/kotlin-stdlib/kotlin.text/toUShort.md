

# toUShort

Parses the string as a UShort number and returns the result.

```kotlin
fun String.toUShort(): UShort(source)
```

```kotlin
fun main() {
    val stringValue = "12345"
    val unsignedShort = stringValue.toUShort()

    println("Original string: $stringValue")
    println("Parsed UShort: $unsignedShort")
}
```
