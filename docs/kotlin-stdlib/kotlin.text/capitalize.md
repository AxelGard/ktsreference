

# capitalize

Warning since 1.5

```kotlin
expect fun String.capitalize(): String(source)
```

```kotlin
fun main() {
    val original = "kotlin is fun"
    val capitalized = original.capitalize()
    println(capitalized) // Output: Kotlin is fun
}
```
