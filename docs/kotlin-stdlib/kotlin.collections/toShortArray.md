

# toShortArray

Returns an array of Short containing all of the elements of this generic array.

```kotlin
fun Array<out Short>.toShortArray(): ShortArray(source)
```

```kotlin
val shortArray = arrayOf<Short>(10, 20, 30).toShortArray()
println(shortArray.contentToString())
```
