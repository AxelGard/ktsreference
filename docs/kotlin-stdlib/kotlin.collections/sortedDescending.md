

# sortedDescending

Returns a list of all elements sorted descending according to their natural sort order.

```kotlin
fun <T : Comparable<T>> Array<out T>.sortedDescending(): List<T>(source)
```

```kotlin
val numbers = arrayOf(10, 2, 8, 4, 6)
println(numbers.sortedDescending())
```
