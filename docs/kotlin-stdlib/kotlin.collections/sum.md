

# sum

Returns the sum of all elements in the array.

```kotlin
@JvmName(name = "sumOfByte")fun Array<out Byte>.sum(): Int(source)
```

```kotlin
val bytes = byteArrayOf(1, 2, 3, 4)
val total = bytes.sum()
println(total)   // prints 10
```
