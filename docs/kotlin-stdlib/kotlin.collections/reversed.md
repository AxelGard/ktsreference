

# reversed

Returns a list with elements in reversed order.

```kotlin
fun <T> Array<out T>.reversed(): List<T>(source)
```

```kotlin
val words = arrayOf("kotlin", "is", "fun")
val reversedWords = words.reversed()
println(reversedWords)  // Output: [fun, is, kotlin]
```
