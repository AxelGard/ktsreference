

# first

Returns the first element.

```kotlin
fun <T> Array<out T>.first(): T(source)
```

```kotlin
fun main() {
    val words = arrayOf("apple", "banana", "cherry")
    val firstWord = words.first()
    println("The first word is: $firstWord")
}
```
