

# max

Returns the largest element.

```kotlin
@JvmName(name = "maxOrThrow")fun Array<out Double>.max(): Double(source)
```

```kotlin
val numbers = arrayOf(4.2, 7.8, 2.5, 9.1, 3.6)
val largest = numbers.max()   // uses Array<Double>.max()
println("The largest number is $largest")
```
