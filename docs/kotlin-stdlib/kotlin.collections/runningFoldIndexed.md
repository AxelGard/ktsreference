

# runningFoldIndexed

Returns a list containing successive accumulation values generated by applying operation from left to right to each element, its index in the original array and current accumulator value that starts with initial value.

```kotlin
inline fun <T, R> Array<out T>.runningFoldIndexed(initial: R, operation: (index: Int, acc: R, T) -> R): List<R>(source)
```

```kotlin
val numbers = arrayOf(1, 2, 3, 4)

val result = numbers.runningFoldIndexed(0) { index, acc, value ->
    acc + index + value
}

println(result)   // [0, 1, 4, 9, 16]
```
