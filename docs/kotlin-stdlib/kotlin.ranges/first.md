

# first

Returns the first element.

```kotlin
fun IntProgression.first(): Int(source)
```

```kotlin
fun main() {
    val range = 1..10
    println(range.first())      // prints 1

    val descending = 10 downTo 1
    println(descending.first()) // prints 10
}
```
