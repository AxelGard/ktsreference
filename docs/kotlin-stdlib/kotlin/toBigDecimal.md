

# toBigDecimal

Returns the value of this Int number as a BigDecimal.

```kotlin
inline fun Int.toBigDecimal(): BigDecimal(source)
```

```kotlin
val intValue = 42
val bigDecimalValue = intValue.toBigDecimal()
println("BigDecimal value: $bigDecimalValue")  // Prints: BigDecimal value: 42
```
