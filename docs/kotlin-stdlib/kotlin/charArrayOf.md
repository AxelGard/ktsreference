

# charArrayOf

Returns an array containing the specified characters.

```kotlin
expect fun charArrayOf(vararg elements: Char): CharArray(source)
```

```kotlin
val chars = charArrayOf('a', 'b', 'c')
println(chars.size)   // 3
println(chars[0])     // a
```
