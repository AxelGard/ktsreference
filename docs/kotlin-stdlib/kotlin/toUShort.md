

# toUShort

Converts this Byte value to UShort.

```kotlin
inline fun Byte.toUShort(): UShort(source)
```

```kotlin
fun main() {
    val b: Byte = 120
    val u: UShort = b.toUShort()
    println("Converted UShort value: $u")  // Output: Converted UShort value: 120
}
```
