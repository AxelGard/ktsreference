

# toULong

Converts this Byte value to ULong.

```kotlin
inline fun Byte.toULong(): ULong(source)
```

```kotlin
fun main() {
    val byteValue: Byte = 42
    val uLongValue: ULong = byteValue.toULong()
    println("ULong value: $uLongValue")  // Output: ULong value: 42
}
```
