

# longArrayOf

Returns an array containing the specified Long numbers.

```kotlin
expect fun longArrayOf(vararg elements: Long): LongArray(source)
```

```kotlin
fun main() {
    val longArray = longArrayOf(100L, 200L, 300L, 400L)
    println(longArray.contentToString())
}
```
