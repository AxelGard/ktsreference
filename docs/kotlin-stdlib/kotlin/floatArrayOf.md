

# floatArrayOf

Returns an array containing the specified Float numbers.

```kotlin
expect fun floatArrayOf(vararg elements: Float): FloatArray(source)
```

```kotlin
val floatArray = floatArrayOf(1.0f, 2.5f, 3.0f)
println(floatArray.joinToString())
```
