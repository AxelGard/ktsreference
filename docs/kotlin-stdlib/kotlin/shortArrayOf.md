

# shortArrayOf

Returns an array containing the specified Short numbers.

```kotlin
expect fun shortArrayOf(vararg elements: Short): ShortArray(source)
```

```kotlin
val shortArray = shortArrayOf(5s, 10s, 15s)
println(shortArray.joinToString())
```
