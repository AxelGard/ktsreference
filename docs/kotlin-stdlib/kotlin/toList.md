

# toList

Converts this pair into a list.

```kotlin
fun <T> Pair<T, T>.toList(): List<T>(source)
```

```kotlin
val pair = Pair("Kotlin", "toList")
val list = pair.toList()
println(list)   // Output: [Kotlin, toList]
```
