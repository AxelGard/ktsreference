# Kotlin Reffrences

This is an alternative set of documentation for Kotlin, designed to be simpler and easier to use than traditional resources.

The examples included here are generated with the help of a large language model (LLM). While care has been taken to ensure accuracy, there may still be mistakes.

If you find any issues or inaccuracies, please feel free to open a pull request with your corrections. Contributions are always welcome!


The source can be find on [GitHub](https://github.com/AxelGard/ktsreference)
